### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Change Password
3. Personal Information
4. Announcement Management
5. Sales Order Management
6. Salary Management


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Basic Data Management (Department Management, Announcement Type Management, Income and Expense Type Management, Sales Type Management, Position Management)
4. Announcement Management
5. Income and Expense Management
6. Sales Order Management
7. Salary Management
8. Employee Management
